Controls

Runner

The runner can be controlled using the "WASD" keys, each arrow representing movement in a direction.


The runner can use a shield to dash and protect himself from attacking enemies, he can do so by pressing the "S" key.

Protector

The protector can be controlled using the "Arrow" keys, each key representing movement in a direction.


The protector can can attack the enemies, he can do so by pressing the "Arrow Down" key.
